# Mundo Automotor - Nissan GT-R

Este es un proyecto sencillo que presenta información sobre el Nissan GT-R, un automóvil deportivo de alto rendimiento. La página web está escrita en HTML y CSS.

## Contenido

- Inicio: Información sobre el origen de Nissan y la historia del Nissan Skyline.
- Nissan GT-R: Detalles sobre el origen y la evolución del Nissan GT-R.
- Contacto: Información de contacto.
